**Tracks codebase state for AI agents and developers.**

## Files

1. **[Implementation Status](Implementation%20Status.md)** - What's implemented

---

## Quick Onboarding (AI Agents)

1. Read Implementation Status → Know what exists
2. Read Script Interactions → Understand connections
3. Use Quick Reference → Look up APIs while coding

**Time:** ~3 minutes

---

## Update Rules

**Always update when:**
- New script added (update Implementation Status)
- Script communication changed (update Script Interactions)
- Public API changed (update Quick Reference)

**Don't update for:**
- Internal refactoring
- Bug fixes without behavior change

---

## Relationship to GDD

- **GDD:** Design intent (what/why)
- **Progress:** Implementation state (what exists)
- **Code comments:** Implementation details

Keep Progress docs **concise and factual**.
